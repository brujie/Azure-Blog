# 力扣算法初级

数组去重

代码：
let nums = [1,1,2];
let removeDuplicates = function(nums) {
    let tempArr = [];
    for(let i=0;i<nums.length;i++){
        let flag = tempArr.indexOf(nums[i]);
        if(flag == -1){
            tempArr.push(nums[i]);
        }
    }
    return tempArr;
};
removeDuplicates(nums);

旋转数组
给你一个数组，将数组中的元素向右轮转 k 个位置，其中 k 是非负数。

示例 1:

输入: nums = [1,2,3,4,5,6,7], k = 3
输出: [5,6,7,1,2,3,4]
解释:
向右轮转 1 步: [7,1,2,3,4,5,6]
向右轮转 2 步: [6,7,1,2,3,4,5]
向右轮转 3 步: [5,6,7,1,2,3,4]
示例 2:

输入：nums = [-1,-100,3,99], k = 2
输出：[3,99,-1,-100]
解释: 
向右轮转 1 步: [99,-1,-100,3]
向右轮转 2 步: [3,99,-1,-100]

代码:
let  nums = [1,2,3,4,5,6,7]
let rotate = function(nums, k) {
    if(nums.length == k){l
        nums.reverse();
    }

    temp = temp.slice(-k);
    nums.length = nums.length - k;
    return [...temp,...nums];

rotate(nums,3)

存在重复元素

给定一个整数数组，判断是否存在重复元素。

如果存在一值在数组中出现至少两次，函数返回 true 。如果数组中每个元素都不相同，则返回 false 。

示例 1:

输入: [1,2,3,1]
输出: true
示例 2:

输入: [1,2,3,4]
输出: false
示例 3:

输入: [1,1,1,3,3,4,3,2,4,2]
输出: true

代码：
let arr = [1,1,1,3,3,4,3,2,4,2];
let containsDuplicate = function(nums) {
    for(let i = 0; i < nums.length - 1; i++) {
        if (nums[i] == nums[i + 1]) {
            return true;
        }
    }
    return false;
}
containsDuplicate(arr)


只出现一次的数字
给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

说明：

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

示例 1:

输入: [2,2,1]
输出: 1
示例 2:

输入: [4,1,2,1,2]
输出: 4

let arr =  [4,1,2,1,2];

var singleNumber = function(arr,k) {
    let tempArr = [];
    for(let i=0;i<arr.length;i++){
        if(arr[i] == k){
            tempArr.push(arr[i]);
        } 
    }
    if(tempArr.length == 1){
        return tempArr[0];
    } else {
        return false;
    }
};
singleNumber(arr,4)
