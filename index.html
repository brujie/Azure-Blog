    # 个人博客搭建

    # 1.使用vuepress

    # 快速上手

    # https://vuepress.vuejs.org/zh/guide/getting-started.html

    # 2.搭建文件目录

    |--docs // 当前目录
├─.vuepress 
│  ├─dist   // 打包生成的dist目录
│  │  ├─assets
│  │  │  ├─css
│  │  │  ├─img
│  │  │  └─js
│  │  └─docs
│  └─public
|  |-- config.js // 配置文件
|  |-- deploy.bat // 可执行文件(非必须)
└─note

    # 3.打包生成dist目录,部署到GitHub Pages

    # 部署 详细的查看官网部署文章,我这里介绍我使用的方法

    # https://vuepress.vuejs.org/zh/guide/deploy.html#%E4%BA%91%E5%BC%80%E5%8F%91-cloudbase

        // 由于官网的.sh文件执行出错,我使用的是bat文件执行
    // 提前github新建一个自己的仓库,名字要与config.js文件中的base对应

    config.js

    module.exports = {
    title:"Azure-Blog",     // 网页标题可自定义
    base:"/Azure-Blog/",    // 名称与github新建的仓库名称必须保持一致
        themeConfig: {
        nav:[
            // 导航
            {
                text:'Github',
                link:'https://github.com/brujie/AzureUI', // 自由设置
            },
        ],
        sidebar: [
            // 侧边栏文章
        ]
        },
    }

    deploy.bat
    
    @echo off
    echo *.bat>.gitignore  
    @title bat 新建gitignore文件...
    set github_link="此处设置变量为自己的github仓库地址"
    @title bat 交互执行git命令...
    git init 
    git remote add origin %github_link%
    git add -A
    git commit -m 'deploy'
    git push -f  %github_link% master:gh-pages
    pause

    # 4.发布成功查看效果

    1.登录自己仓库查看是否上传成功 2.点击settings,找到侧边栏pages选项 3.点击查看自己的博客地址
